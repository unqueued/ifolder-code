Client iFolder

Il client iFolder si integra con il sistema operativo per fornire i servizi iFolder nell'ambiente desktop nativo. Il client iFolder consente di gestire le proprie cartelle iFolder e di selezionare le cartelle iFolder condivise da configurare su ciascun computer. iFolder supporta i seguenti sistemi operativi:

  • SUSE Linux Enterprise Desktop (SLED) 10, SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2 a 64 bit

  • openSUSE 12.1

  • Windows XP SP3 a 32 bit

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X a 32 bit (architettura Intel) v10.6.7 e successive (richiede Mono 2.4.2.3). L'architettura PowerPc non Ã¨ supportata.
