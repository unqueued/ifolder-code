Client iFolder

Intégré à votre système d'exploitation, le client iFolder fournit des services iFolder dans votre environnement de bureau natif. Le client iFolder vous permet de gérer vos propres dossiers iFolder et de sélectionner quels dossiers iFolder partagés vous devez configurer sur chaque ordinateur. iFolder prend en charge les systèmes d'exploitation suivants :

  • SUSE Linux Enterprise Desktop (SLED) 10 , SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP1 64 bits

  • openSUSE 11.4

    Le client iFolder Linux requiert Mono pour Linux et la prise en charge du plug-in Nautilus pour iFolder du bureau GNOME.

  • Windows XP SP3 32 bits

  • Windows Vista SP1

  • Windows 7

  • Macintosh OS X 32 bits (architecture Intel) v10.5 et version ultérieure (requiert Mono 2.4.2.3). L'architecture PowerPc n'est pas prise en charge.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
