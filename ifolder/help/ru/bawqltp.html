Клиент iFolder

Клиент iFolder интегрируется с операционной системой и предоставляет сервисы iFolder в соответствующей вычислительной среде рабочей станции. Клиент iFolder позволяет управлять собственными папками iFolder и выбрать, какие совместно используемые папки iFolder необходимо установить на каждом компьютере. Продукт iFolder поддерживает следующие операционные системы:

  • SUSE Linux Enterprise Desktop (SLED) 10 с пакетом обновления 3

  • 64-разрядная версия SUSE Linux Enterprise Desktop (SLED) 10 с пакетом обновления 2

  • openSUSE 12.1

  • 32-разрядная версия Windows XP с пакетом обновления 3

  • Windows Vista с пакетом обновления 2

  • Windows 7 с пакетом обновления 1 (SP1)

  • 32-разрядная версия Macintosh OS X (на архитектуре Intel) v10.6.7 и более поздних версий (требует Mono 2.4.2.3). Архитектура PowerPc не поддерживается.

Дополнительную информацию см. в Юридическая информация.
