Klient aplikácie iFolder

Klient aplikácie iFolder je začlenený do operačného systému, aby mohol poskytovať služby aplikácie iFolder v pracovnom prostredí vášho vlastného počítača. Klient aplikácie iFolder umožňuje spravovať vlastné priečinky iFolder a vybrať zdieľanie rôznych priečinkov iFolder na jednotlivých počítačoch. Aplikácia iFolder podporuje nasledujúce operačné systémy:

  • SUSE Linux Enterprise Desktop (SLED) 10 SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP1 (64-bitová verzia)

  • openSUSE 11.4

    Klient aplikácie iFolder pre systém Linux vyžaduje rámec Mono pre systém Linux a podporu doplnku GNOME desktop for iFolder Nautilus.

  • Windows XP SP3 (32-bitová verzia)

  • Windows Vista SP1

  • systém Windows 7

  • Macintosh OS X (32-bitová verzia) (architektúra Intel) v10.5 a novšie (vyžaduje rámec Mono 2.4.2.3). Architektúra PowerPc nie je podporovaná.

Informácie o ochranných známkach a autorských právach nájdete v časti Právne prehlásenia.
