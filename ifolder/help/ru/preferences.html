Конфигурирование настроек iFolder для клиента

  1. В области оповещений щелкните правой кнопкой значок Сервисы iFolder , затем выберите Настройки, чтобы открыть диалоговое окно "Настройки iFolder" на вкладке Общие.

  2. Настройте следующие параметры:

    Параметр

    Описание

    Запуск iFolder при входе на рабочую станцию (только в версии клиента Windows)

    Установите этот флажок, чтобы клиент iFolder запускался при входе локального пользователя, с учетной записью которого устанавливался клиент iFolder. Для учетных записей с установленным параметром Запомнить пароль, пароль при входе не запрашивается.

    Для запуска iFolder вручную снимите этот флажок.

    Показ диалогового окна с подтверждением при успешном создании папки iFolder

    Установите этот флажок, чтобы при создании каждой новой папки iFolder в дополнение к обычным визуальным эффектам (таким как появление папки iFolder в списке папок) получать сообщение с подтверждением.

    Снимите этот флажок, чтобы отключить сообщения с подтверждением.

    Скрыть значок iFolder на панели задач (доступно только для клиента Windows)

    Чтобы скрыть значок iFolder на панели задач, установите этот флажок.

    Чтобы показывать значок iFolder на панели задач, установите этот флажок.

    ПРЕДУПРЕЖДЕНИЕ.При установке данного флажка убедитесь, что флажок Скрывать главное окно iFolder при запуске снят. В противном случае визуальные подсказки, указывающие на выполнение iFolder, будут недоступны.

    Скрывать главное окно iFolder при запуске

    Установите этот флажок, чтобы отключить показ окна iFolder при запуске.

    Снимите этот флажок, чтобы включить показ окна iFolder при запуске.

    Показывать журналы синхронизации

    Установите этот флажок, чтобы включить показ журналов синхронизации при запуске.

    Снимите этот флажок, чтобы отключить показ журналов синхронизации при запуске.

    При нарушении политики квот

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений при каждом нарушении политики дисковых квот.

    Чтобы отключить это оповещение, снимите этот флажок.

    При нарушении политики в отношении размера файла

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений при каждом нарушении политики в отношении размера файла.

    Чтобы отключить это оповещение, снимите этот флажок.

    При нарушении политики исключения файлов

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений при каждом нарушении политики исключения файлов.

    Чтобы отключить это оповещение, снимите этот флажок.

    Когда диск полон

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений, когда диск в локальной системе заполнен.

    Чтобы отключить это оповещение, снимите этот флажок.

    Если необходимые разрешения недоступны

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений каждый раз, когда разрешения на выполнение операции с файлом или папкой недоступны. Например, невозможно загрузить файл при наличии только прав на чтение.

    Чтобы отключить это оповещение, снимите этот флажок.

    Если путь к файлу превышает оптимальное ограничение

    Установите этот флажок, чтобы показывать всплывающие сообщения в области оповещений каждый раз, когда длина пути к файлу превышает максимальную длину, определенную в системе. Например, на платформах на базе Windows количество символов пути не должно превышать 248.

    Чтобы отключить это оповещение, снимите этот флажок.

    При совместном использовании папок iFolder

    Установите этот флажок, чтобы включить показ всплывающих сообщений в области уведомлений при получении приглашения к совместному использованию папки iFolder.

    Чтобы отключить это оповещение, снимите этот флажок.

    При возникновении конфликтов

    Установите этот флажок, чтобы включить показ всплывающих сообщений в области уведомлений в случае возникновении конфликтов при синхронизации файлов.

    Снимите этот флажок. чтобы отключить всплывающие сообщения о конфликтах при синхронизации.

    При возникновении конфликтов в папке iFolder отображается значок ошибок iFolder .

    Синхронизация

    Автоматически синхронизировать папки iFolder. Установите этот флажок, чтобы включить синхронизацию всех папок iFolder на данном компьютере при Вашей регистрации или входе в систему с текущими идентификационными данными. Снимите этот флажок, чтобы отключить синхронизацию.

    Интервал синхронизации папок iFolder. Укажите минимальный интервал, который будет использоваться для синхронизации папок iFolder на данном компьютере, на котором используется текущая учетная запись. Укажите значение и единицы измерения. Если эта опция включена, значение по умолчанию составляет 5 минут. Минимально допустимый интервал — 5 секунд.

    Если клиент iFolder используется с сервером Novell iFolder 3.7 или более поздней версии, действующим минимальным интервалом синхронизации всегда является наибольшее значение из следующих:

    • Системная политика по умолчанию имеет нулевое значение (0), если не установлена пользовательская политика. Если установлена пользовательская политика, она переопределяет системную политику, независимо от величины значения.

    • Политика локального компьютера или настройка синхронизации с сервером на клиентской рабочей станции.

    • Политика iFolder (совокупная).

  3. Щелкните Закрыть, чтобы закрыть диалоговое окно настроек iFolder.

Дополнительную информацию см. в Юридическая информация.
