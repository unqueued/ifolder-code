Klient iFolder

Klient iFolder se integruje s operačním systémem a poskytuje služby aplikace iFolder v přirozeném prostřední vaší pracovní plochy. Klient iFolder umožňuje provádět správu složek iFolder a vybrat, které sdílené složky iFolder budou v každém počítači nastaveny. iFolder podporuje následující operační systémy:

  • SUSE Linux Enterprise Desktop (SLED) 10 SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2 64bitový

  • openSUSE 12.1

  • Windows XP SP3 32bitový

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X 32bitový (pro procesory Intel) verze 10.6.7 a vyšší (vyžaduje prostředí Mono 2.4.2.3). Procesory PowerPC nejsou podporovány.

Další informace získáte v části Právní upozornění.
