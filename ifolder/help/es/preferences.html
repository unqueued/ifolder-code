Configuración de las preferencias de iFolder del cliente

  1. Haga clic con el botón derecho en el icono Servicios de iFolder en el área de notificaciones y, a continuación, seleccione Preferencias para abrir la pestaña General del recuadro de diálogo Preferencias de iFolder.

  2. Especifique sus preferencias para las siguientes opciones:

    Parámetro

    Descripción

    Iniciar iFolder al entrar en el escritorio (sólo disponible en el Cliente Windows)

    Seleccione este recuadro de verificación para iniciar iFolder siempre entre a la sesión con el nombre de usuario local en el equipo donde ha configurado la cuenta de iFolder. En las cuentas para las que haya habilitado Recordar mi contraseña, no se le pedirá que introduzca su contraseña al entrar a la sesión.

    Deseleccione este recuadro de verificación para iniciar iFolder manualmente.

    Visualizar recuadro de diálogo de confirmación sobre la creación satisfactoria de iFolder

    Seleccione este recuadro de verificación para recibir un mensaje de confirmación cada vez que cree una carpeta iFolder además de las indicaciones visuales habituales como, por ejemplo, la aparición de la carpeta iFolder en la lista de carpetas iFolder.

    Deseleccione este recuadro de verificación para dejar de recibir el mensaje de confirmación.

    Ocultar el icono de bandeja de iFolder (disponible sólo en cliente Windows)

    Seleccione este recuadro de verificación para ocultar el icono de iFolder en la bandeja del sistema.

    Deseleccione este recuadro de verificación para mostrar el icono de iFolder en la bandeja del sistema.

    ADVERTENCIA:Si selecciona este recuadro de verificación, asegúrese de que esté deseleccionado el recuadro Ocultar ventana principal de iFolder en el inicio. De lo contrario, no estarán disponibles las indicaciones visuales que avisan de que hay una aplicación de iFolder en ejecución.

    Ocultar ventana principal de iFolder durante el inicio

    Seleccione este recuadro de verificación si desea que la ventana de iFolder permanezca oculta durante el inicio.

    Deseleccione este recuadro de verificación para visualizar la ventana de iFolder durante el inicio.

    Visualizar registros de sincronización

    Seleccione este recuadro de verificación si desea visualizar los registros de sincronización durante el inicio.

    Deseleccione este recuadro de verificación si desea ocultar los registros de sincronización durante el inicio.

    Cuando se infringe una directiva de cuotas

    Deseleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se infrinja una directiva de cuota de disco.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando se infringe una directiva sobre tamaño de archivos

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se infrinja una directiva sobre tamaño de archivos.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando se infringe una directiva de exclusión de archivos

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se infrinja una directiva de exclusión de archivos.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando el disco está lleno

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que el disco esté lleno en el sistema local.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando los permisos necesarios no están disponibles

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que no estén disponibles los permisos necesarios para realizar una operación en un archivo o una carpeta. Por ejemplo, no podrá descargar un archivo si sólo tiene derechos de lectura.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando la ruta de archivo supera el límite óptimo

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que una ruta de archivo supere la longitud máxima definida por el sistema. Por ejemplo, en las plataformas basadas en Windows, el número de caracteres de una ruta debe ser menor de 248.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando las iFolders están compartidas

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que reciba una nueva invitación para compartir una carpeta iFolder.

    Deseleccione este recuadro de verificación para inhabilitar esta notificación.

    Cuando surgen conflictos

    Seleccione este recuadro de verificación para visualizar un mensaje emergente en el área de notificaciones siempre que se produzca un conflicto al sincronizar archivos.

    Deseleccione este recuadro de verificación para inhabilitar los mensajes emergentes en caso de conflicto de sincronización.

    Cuando se produzca algún conflicto, se mostrará el icono Error de iFolder en la carpeta iFolder en cuestión.

    Sincronización

    Automatically Synchronize iFolders (Sincronizar carpetas iFolder automáticamente): Seleccione esta casilla de verificación para habilitar la sincronización en todas las carpetas iFolder del equipo en cuestión con la identidad de entrada o inicio de sesión local actual. Deseleccione la casilla de verificación para inhabilitar la sincronización.

    Synchronize iFolders Every (Sincronizar carpetas iFolder cada): Especifique el intervalo mínimo que desea utilizar para sincronizar carpetas iFolder en este equipo con su identidad de entrada o inicio de sesión local actual. Especifique el valor y las unidades. Si esta opción está habilitada, el tiempo por defecto es 5 minutos. El intervalo mínimo puede fijarse en 5 segundos.

    Si utiliza el Cliente iFolder con un servidor Novell iFolder 3.7 o posterior, el intervalo de sincronización mínimo que se aplicará siempre será el valor más alto de los parámetros siguientes:

    • El valor por defecto de la directiva del sistema es cero (0), a menos que se establezca una directiva de usuario. Si existe una directiva de usuario establecida, ésta anulará la directiva del sistema, tanto si es superior o inferior en valor.

    • La directiva del equipo local o el ajuste del equipo cliente que se sincroniza con el servidor.

    • La directiva de iFolder (recopilación).

  3. Haga clic en Cerrar para cerrar el recuadro de diálogo Preferencias de iFolder.

Para obtener más información, consulte Información legal.
