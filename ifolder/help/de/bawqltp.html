iFolder-Client

Der iFolder-Client wird im Betriebssystem integriert, um iFolder-Dienste in Ihrer Desktop-Umgebung bereitzustellen. Mit dem iFolder-Client können Sie Ihre eigenen iFolder-Ordner verwalten und auswählen, welche freigegebenen iFolder-Ordner auf dem jeweiligen Computer eingerichtet werden. iFolder unterstützt folgende Betriebssysteme:

  • SUSE Linux Enterprise Desktop (SLED) 10 , SP3

  • 64-Bit-SUSE Linux Enterprise Desktop (SLED) 11 SP1

  • openSUSE 11.3

    Für den iFolder-Linux-Client ist das Mono-Framework für Linux und das Nautilus-Plugin für die GNOME Desktop-iFolder-Unterstützung erforderlich.

  • 32-Bit-Windows XP SP3

  • Windows Vista SP1

  • Windows 7

  • 32-Bit-Macintosh OS X (Intel-Architektur) v10.5 und höher (erfordert Mono 2.4.2.3). PowerPc-Architektur wird nicht unterstützt.
