iFolderクライアント

iFolderクライアントは、ネイティブなデスクトップ環境でiFolderサービスを提供するために、オペレーティングシステムを統合します。iFolderクライアントで、自分が所有するiFolderを管理したり、どの共有iFolderを各コンピュータ上にセットアップするかを選択できます。iFolderは次のオペレーティングシステムをサポートしています。

  • SUSE Linux Enterprise Desktop (SLED) 10 、SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2 64ビット

  • openSUSE 12.1

  • Windows XP SP3 32ビット

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X 32ビット(Intelアーキテクチャ)v10.6.7以降(Mono 2.4.2.3が必要)。PowerPcアーキテクチャはサポートされていません。

詳細については、「 保証と著作権.
