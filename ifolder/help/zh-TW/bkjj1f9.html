iFolder 瀏覽器

iFolder 瀏覽器可讓您管理您的 iFolder。下面提供的表格摘述 iFolder 瀏覽器元素：

表 2 iFolder 瀏覽器元素

元素名稱

描述

建立 iFolder

讓您將本地機器上的資料夾轉換為 iFolder，並將它上載到您目前連接的 iFolder 伺服器。

如需詳細資訊，請參閱建立和上傳 iFolder

立即同步化

讓您立即在本地機器與伺服器之間起始同步 iFolder 的作業。

共享

讓您與其他使用者共享您的 iFolder。

如需詳細資訊，請參閱共用 iFolder

恢復為一般資料夾

讓您將 iFolder 恢復為一般資料夾。您也可以從伺服器刪除 iFolder。如需詳細資訊，請參閱將 iFolder 恢復為一般資料夾

伺服器

顯示您已連接之 iFolder 伺服器的 IP 位址。

名稱

顯示所選 iFolder 的名稱。

擁有者

顯示所選 iFolder 的擁有者名稱。

權限

顯示 iFolder 擁有者所擁有的權限。

待同步的檔案/資料夾

指出待同步的檔案或資料夾數目。

上次成功同步時間

指出上次成功同步的時間與日期

下載 iFolder

讓您從 iFolder 伺服器下載 iFolder 到本地機器。

如需詳細資訊，請參閱下載可用的 iFolder

iFolder 類型

指出選取的 iFolder 是加密或一般的 iFolder。

iFolder 大小

顯示所選 iFolder 的大小。

合併

「合併」會協助您統整 iFolder 資料並使 iFolder 伺服器語用戶端之間的資料傳輸降到最低。您可以合併伺服器上的 iFolder 內容與現有本地機器資料夾的內容。如需詳細資訊，請參閱合併 iFolder

從伺服器刪除

讓您從伺服器刪除選取的 iFolder。

帳戶清單

顯示已設定您的帳戶的伺服器清單。

連接/解除連接

讓您連接到 iFolder 伺服器或從 iFolder 伺服器解除連接。若已連接到 iFolder 伺服器，會顯示「解除連接」按鈕；若未連接到任何 iFolder 伺服器，會顯示「連接」按鈕。

解析

讓您解決檔案衝突。若要解決檔案版本的衝突，您必須在「解決衝突」對話方塊中選取正確的版本。如需詳細資訊，請參閱解決檔案衝突

使用者

顯示已登入所選 iFolder 伺服器之使用者的名稱。

iFolder 數目

顯示 iFolder 伺服器上的 iFolder 以及所選 iFolder 伺服器之本地機器上的 iFolder 總數。

磁碟配額

顯示「已使用」與「可用」欄位的值。

已使用

顯示目前所選 iFolder 伺服器上的所有檔案與目錄佔用的伺服器磁碟空間總數。

可用的

顯示目前可供所選 iFolder 伺服器上的檔案或目錄使用的伺服器磁碟空間總數。

自動同步

顯示兩個連續自動同步循環的間隔時間

有關更詳細的資訊，請參閱 法律聲明.
