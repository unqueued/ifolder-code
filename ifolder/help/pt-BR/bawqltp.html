Cliente iFolder

O cliente do iFolder integra-se ao seu sistema operacional para fornecer serviços do iFolder em seu ambiente nativo de área de trabalho. O cliente iFolder permite que você gerencie suas próprias pastas iFolder e selecione quais pastas iFolder compartilhadas deseja configurar em cada computador. O iFolder suporta os seguintes sistemas operacionais:

  • SUSE Linux Enterprise Desktop (SLED) 10 , SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP1 64 bits

  • openSUSE 11.4

    O cliente iFolder da Linux exige a estrutura Mono para Linux e uma área de trabalho GNOME para oferecer suporte ao plug-in Nautilus do iFolder.

  • Windows XP SP3 32 bits

  • Windows Vista SP1

  • Windows 7

  • Macintosh OS X 32 bits (arquitetura Intel) v10.5 e posteriores (exige o Mono 2.4.2.3). A arquitetura PowerPC não é suportada.

Para obter mais informações, consulte Informações legais.
