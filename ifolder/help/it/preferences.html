Configurazione delle preferenze di iFolder per il client

  1. Fare clic con il pulsante destro del mouse sull'icona Servizi iFolder nell'area di notifica e scegliere Preferenze per aprire la scheda Generale della finestra di dialogo Preferenze di iFolder.

  2. Specificare le preferenze seguenti:

    Parametro

    Descrizione

    Avviare iFolder quando si esegue il login al desktop (disponibile solo nel client Windows)

    Selezionare questa casella di controllo per avviare iFolder ogni volta che si esegue il login con il nome utente locale utilizzato per configurare l'account di iFolder. Per gli account nei quali è stata abilitata l'opzione Memorizza password, al momento dell'accesso non viene richiesto di specificare la password.

    Per avviare iFolder manualmente, deselezionare questa casella di controllo.

    Visualizza finestra di conferma della creazione della cartella iFolder

    Selezionare questa casella di controllo per ricevere un messaggio di conferma ogni volta che si crea una cartella iFolder, in aggiunta ai normali indicatori visivi, come ad esempio la visualizzazione di iFolder nell'elenco delle cartelle iFolder.

    Deselezionare la casella di controllo per interrompere la ricezione del messaggio di conferma.

    Nascondi icona iFolder della barra delle applicazioni (disponibile solo nei client Windows)

    Selezionare questa casella di controllo per nascondere l'icona iFolder visualizzata nella barra delle applicazioni.

    Deselezionare questa casella di controllo per visualizzare l'icona iFolder nella barra delle applicazioni.

    AVVERTENZA:Se si seleziona questa casella di controllo, assicurarsi che la casella di controllo Nascondi finestra principale di iFolder all'avvio sia deselezionata. In caso contrario, i segnali visivi indicanti un'applicazione iFolder in esecuzione non sono disponibili.

    Nascondi la finestra principale di iFolder all'avvio

    Selezionare questa casella di controllo se si desidera che la finestra iFolder rimanga nascosta all'avvio.

    Deselezionare la casella di controllo per visualizzare la finestra iFolder all'avvio.

    Visualizza log di sincronizzazione

    Selezionare questa casella di controllo per visualizzare i log di sincronizzazione all'avvio.

    Deselezionare questa casella di controllo per nascondere i log di sincronizzazione all'avvio.

    Quando viene violata una norma della quota

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che viene violata una norma della quota disco.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando è violata la norma della dimensione del file

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che viene violata una norma della dimensione del file.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando è violata la norma di esclusione del file

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che viene violata una norma di esclusione del file.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando il disco è pieno

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che il disco è pieno nel sistema locale.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando non sono disponibili le autorizzazioni necessarie

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che le autorizzazioni necessarie per eseguire un'operazione su un file o una cartella non sono disponibili. Ad esempio, se si dispone dei diritti di sola lettura non è possibile eseguire il download di un file.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando il percorso del file supera il limite ottimale

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che il percorso di un file supera la lunghezza massima definita dal sistema. Ad esempio, sulle piattaforme basate su Windows, il numero di caratteri presenti in un percorso deve essere inferiore a 248.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando vengono condivise le cartelle iFolder

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che si riceve un nuovo invito a condividere una cartella iFolder.

    Deselezionare questa casella di controllo per disabilitare la notifica.

    Quando si verifica un conflitto

    Selezionare questa casella di controllo per visualizzare un messaggio popup nell'area di notifica ogni volta che si verifica un conflitto con i file di sincronizzazione.

    Deselezionare questa casella di controllo per disabilitare i messaggi popup per i conflitti di sincronizzazione.

    Ogni volta che si verifica un conflitto, l'icona Errore di iFolder viene visualizzata nella cartella iFolder.

    Sincronizzazione

    Sincronizza cartelle iFolder automaticamente: Selezionare questa casella di controllo per abilitare la sincronizzazione di tutte le cartelle iFolder del computer associate all'identità di login locale attuale. Deselezionare la casella di controllo per disabilitare la sincronizzazione.

    Sincronizza cartelle iFolder ogni: Specificare l'intervallo minimo per la sincronizzazione delle cartelle iFolder disponibili nel computer con l'identità di login locale attuale. Specificare il valore e le unità. Se l'opzione è abilitata, il valore di default è pari a 5 minuti. L'intervallo minimo può essere impostato su 5 secondi.

    Se si utilizza il client iFolder con il server Novell iFolder 3.7 o versione successiva, l'intervallo minimo di sincronizzazione effettivo corrisponde sempre al valore maggiore tra quelli delle seguenti impostazioni:

    • Il valore di default delle norme del sistema è zero (0), a meno che non vengano impostate norme utente. In questo caso, le norme utente hanno priorità su quelle di sistema indipendentemente dal fatto che il relativo valore sia maggiore o minore.

    • Le norme del computer locale oppure l'impostazione nel computer client che esegue la sincronizzazione con il server.

    • Le norme di iFolder (raccolta).

  3. Fare clic su Chiudi per chiudere la finestra di dialogo Preferenze di iFolder.

Per ulteriori informazioni, vedere Note legali.
