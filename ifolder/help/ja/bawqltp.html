iFolderクライアント

iFolderクライアントは、ネイティブなデスクトップ環境でiFolderサービスを提供するために、オペレーティングシステムを統合します。iFolderクライアントで、自分が所有するiFolderを管理したり、どの共有iFolderを各コンピュータ上にセットアップするかを選択できます。iFolderは次のオペレーティングシステムをサポートしています。

  • SUSE Linux Enterprise Desktop (SLED) 10 、SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP1 64ビット

  • openSUSE 11.4

    iFolder Linuxクライアントには、Linux用MonoフレームワークおよびGNOMEデスクトップのforiFolder Nautilusプラグインのサポートが必要です。

  • Windows XP SP3 32ビット

  • Windows Vista SP1

  • Windows 7

  • Macintosh OS X 32ビット(Intelアーキテクチャ)v10.5以降(Mono 2.4.2.3が必要)。PowerPcアーキテクチャはサポートされていません。

詳細については、「 保証と著作権.
