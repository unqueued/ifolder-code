Configuration des préférences iFolder du Client

  1. Cliquez avec le bouton droit sur l'icône Services iFolder dans la zone Notification, puis sélectionnez Préférences pour ouvrir la boîte de dialogue Préférences iFolder sous l'onglet Général.

  2. Spécifiez vos préférences pour les paramètres suivants :

    Paramètre

    Description

    Démarrer iFolder lors du login au bureau (uniquement disponible pour le client Windows)

    Cochez cette case pour lancer iFolder lorsque vous vous loguez avec le nom d'utilisateur local que vous avez défini lors de la création du compte iFolder. Lorsque vous vous loguez aux comptes pour lesquels vous avez activé l'option Mémoriser mon mot de passe, vous n'êtes pas invité à spécifier votre mot de passe.

    Décochez cette case pour démarrer iFolder manuellement.

    Afficher la boîte de dialogue de confirmation de la création d'un dossier iFolder

    Cochez cette case pour recevoir un message de confirmation à chaque fois que vous créez un dossier iFolder, en plus des indicateurs visuels habituels comme l'apparition du dossier iFolder dans la liste correspondante.

    Désélectionnez cette case pour ne plus recevoir le message de confirmation.

    Masquer l'icône d'iFolder dans la barre système (disponible uniquement avec le client Windows)

    Cochez cette case pour masquer l'icône iFolder affichée dans la barre système.

    Décochez cette case pour afficher l'icône iFolder dans la barre système.

    AVERTISSEMENT :Si vous cochez cette case, veillez à décocher la case Masquer la fenêtre principale de iFolder au démarrage, sinon les indicateurs visuels signalant qu'une application iFolder est en cours d'exécution ne sont pas disponibles.

    Masquer la fenêtre principale du dossier iFolder au démarrage

    Cochez cette case si vous souhaitez que la fenêtre iFolder reste masquée au démarrage.

    Désélectionnez cette case à cocher pour afficher la fenêtre iFolder au démarrage.

    Afficher les journaux de synchronisation

    Cochez cette case si vous souhaitez afficher les journaux de synchronisation au démarrage.

    Désélectionnez cette case à cocher si vous souhaitez masquer les journaux de synchronisation au démarrage.

    Lorsqu'une stratégie de quota est violée

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'une stratégie de quota de disque est violée.

    Décochez cette case pour désactiver cette notification.

    Lorsqu'une stratégie de taille de fichier est violée

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'une stratégie de taille de fichier est violée.

    Décochez cette case pour désactiver cette notification.

    Lorsqu'une stratégie d'exclusion de fichier est violée

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'une stratégie d'exclusion de fichier est violée.

    Décochez cette case pour désactiver cette notification.

    Lorsque le disque est plein

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsque le disque est plein sur le système local.

    Décochez cette case pour désactiver cette notification.

    Lorsque les autorisations requises sont indisponibles

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsque les autorisations requises pour effectuer une opération sur un fichier ou un dossier ne sont pas disponibles. Par exemple, vous ne pouvez pas télécharger un fichier si vous ne disposez que de droits de lecture.

    Décochez cette case pour désactiver cette notification.

    Lorsque le chemin du fichier dépasse la taille optimale

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'un chemin de fichier dépasse la longueur maximale définie par le système. Par exemple, sur les plates-formes Windows, le nombre de caractères compris dans un chemin doit être inférieur à 248.

    Décochez cette case pour désactiver cette notification.

    Lorsque des dossiers iFolder sont partagés

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsque vous recevez une nouvelle invitation à partager un dossier iFolder.

    Décochez cette case pour désactiver cette notification.

    Lorsque des conflits surviennent

    Cochez cette case pour afficher un message contextuel dans la zone de notification lorsqu'un conflit se produit lors de la synchronisation des fichiers.

    Désélectionnez-la pour désactiver les messages contextuels en cas de conflits de synchronisation.

    En cas de conflits, l'icône Erreur iFolder s'affiche sur le dossier iFolder.

    Synchronisation

    Synchroniser automatiquement les dossiers iFolder : cochez cette case pour activer la synchronisation de tous les dossiers iFolder sur cet ordinateur sous votre identité de login ou votre login local actuel. Désélectionnez la case pour désactiver la synchronisation.

    Synchroniser les dossiers iFolder toutes les : spécifiez l'intervalle minimum à utiliser pour synchroniser les dossiers iFolder sur cet ordinateur sous votre identité de login ou votre login actuel local. Spécifiez la valeur et les unités. Si cette option est activée, la valeur par défaut est de 5 minutes. L'intervalle minimum peut être défini sur 5 secondes.

    Si vous utilisez le client iFolder avec un serveur Novell iFolder 3.7 ou version ultérieure, l'intervalle de synchronisation minimal effectif est toujours la valeur la plus élevée des paramètres suivants :

    • La valeur par défaut de la stratégie système est zéro (0), sauf si une stratégie utilisateur est définie. Si c'est le cas, la stratégie d'utilisateur prévaut sur la stratégie du système, que la valeur de la stratégie d'utilisateur soit plus grande ou plus petite que la stratégie du système.

    • La stratégie de la machine locale ou la configuration sur la machine du client synchronisée avec le serveur.

    • La stratégie (de collection) du dossier iFolder.

  3. Cliquez sur Fermer pour fermer la boîte de dialogue Préférences iFolder.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
