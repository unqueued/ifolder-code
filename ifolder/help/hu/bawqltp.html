Az iFolder ügyfél

Az iFolder ügyfél az operációs rendszerbe beépülve, az eredeti asztali környezetben nyújtja az iFolder szolgáltatásait. Az iFolder ügyfél segítségével kezelheti saját iFolder mappáit, és kiválaszthatja, hogy melyik számítógépen mely iFolder mappák használatát állítja be. Az iFolder a következő operációs rendszereken használható:

  • SUSE Linux Enterprise Desktop (SLED) 10 SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2, 64 bites változat

  • openSUSE 12.1

  • Windows XP SP3, 32 bites változat

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X v10.6.7 vagy újabb verzió, 32 bites változat (Intel-architektúra) és a Mono 2.4.2.3. A PowerPC-architektúra nem támogatott.
