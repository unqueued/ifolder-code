为客户程序配置 iFolder 首选设置

  1. 在通知区域中右击“iFolder 服务” 图标，然后选择“首选设置”以打开“iFolder 首选设置”对话框并进入“一般”选项卡。

  2. 指定以下选项的首选设置：

    参数

    说明

    登录到桌面时启动 iFolder（仅在 Windows 客户端中可用）

    选择此复选框可在您使用本地用户名登录到设置 iFolder 帐户的位置时启动 iFolder。对于您启用了“记住我的口令”的帐户，系统不会在您登录时提示您指定口令。

    取消选择此复选框可手动启动 iFolder。

    成功创建 iFolder 时显示确认讯息

    选中此复选框可在每次创建 iFolder 时除了接收常规可视提示（如 iFolder 出现在 iFolder 列表中）外，还接收确认讯息。

    取消选中此复选框可停止接收确认讯息。

    隐藏 iFolder 系统盘图标（仅在 Windows 客户端中可用）

    选择此复选框可隐藏系统盘中显示的 iFolder 图标。

    取消选择此复选框可在系统盘中显示 iFolder 图标。

    警告：如果您选择此复选框，请确保取消选择了“在启动时隐藏 iFolder 主窗口”复选框。否则，指示有个 iFolder 应用程序正在运行的可视提示将不可用。

    启动时隐藏 iFolder 主窗口

    如果希望在启动时隐藏 iFolder 窗口，请选中此复选框。

    取消选中此复选框以便在启动时显示 iFolder 窗口。

    显示同步日志

    如果希望在启动时显示同步日志，请选中此复选框。

    如果希望在启动时隐藏同步日志，请取消选中此复选框。

    如果违反了定额策略

    选择此复选框可在违反磁盘定额策略时，在通知区域中显示弹出讯息。

    取消选择此复选框可禁用此通知。

    如果违反文件大小策略

    选择此复选框可在违反文件大小策略时，在通知区域中显示弹出讯息。

    取消选择此复选框可禁用此通知。

    如果违反文件排除策略

    选择此复选框可在违反文件排除策略时，在通知区域中显示弹出讯息。

    取消选择此复选框可禁用此通知。

    如果磁盘已满

    选择此复选框可在本地系统中的磁盘已满时，在通知区域中显示弹出讯息。

    取消选择此复选框可禁用此通知。

    如果所需权限不可用

    选择此复选框可在对文件或文件夹执行操作所需的权限不可用时，在通知区域中显示弹出讯息。例如，如果您只具有读权限，则无法下载文件。

    取消选择此复选框可禁用此通知。

    如果文件路径超出最佳限制

    选择此复选框可在文件路径超出系统定义的最大长度时，在通知区域中显示弹出讯息。例如，在基于 Windows 的平台上，路径中的字符数必须少于 248。

    取消选择此复选框可禁用此通知。

    如果共享 iFolder

    选中此复选框以便在接收到新的邀请以共享 iFolder 时，在通知区域中显示弹出讯息。

    取消选择此复选框可禁用此通知。

    如果发生冲突

    选中此复选框，以便同步文件时出现冲突时，在通知区域中显示弹出讯息。

    取消选中此复选框以禁用同步冲突的弹出讯息。

    当发生冲突时，iFolder 上会显示 iFolder 错误 图标。

    同步

    Automatically Synchronize iFolders: 选中此复选框可为此计算机上当前本地登录或登录身份下的所有 iFolder 启用同步。取消选中此复选框以禁用同步。

    Synchronize iFolders Every: 指定用于同步此计算机上当前本地登录或登录身份下的 iFolder 的最小间隔。指定值和单位。如果启用了此选项，则默认值为 5 分钟。最小时间间隔可设置为 5 秒。

    如果将此 iFolder 客户程序与 Novell iFolder 3.7 或更高版本的服务器一起使用，则有效的最小同步间隔始终是以下设置的最大值：

    • 系统策略默认为零 (0)，除非已设置用户策略。如果设置了用户策略，无论用户策略的值是大于还是小于系统策略的值，此用户策略都将覆盖系统策略。

    • 本地计算机策略或客户机上的设置与服务器同步。

    • iFolder（集合）策略。

  3. 单击“关闭”来关闭“iFolder 首选设置”对话框。

有关更多信息，请参见 法律声明.
