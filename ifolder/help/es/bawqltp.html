El Cliente iFolder

El Cliente iFolder se integra con el sistema operativo para proporcionar servicios de iFolder en el entorno nativo de su escritorio. El Cliente iFolder permite gestionar las carpetas iFolder y seleccionar las carpetas iFolder compartidas que se van a configurar en cada equipo. iFolder es compatible con los siguientes sistemas operativos:

  • SUSE Linux Enterprise Desktop (SLED) 10, SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2 de 64 bits

  • openSUSE 12.1

  • Windows XP SP3 de 32 bits

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X de 32 bits (arquitectura Intel) v10.6.7 y posteriores (requiere Mono 2.4.2.3). La arquitectura PowerPc no es compatible.

Para obtener más información, consulte Información legal.
