Affichage et configuration des propriétés d'un dossier iFolder

La page Propriétés des dossiers iFolder vous permet d'afficher et de configurer les informations suivantes concernant un dossier iFolder sélectionné.

Présentation des propriétés iFolder

Les paramètres suivants indiquent l'espace disque utilisé sur le serveur :

Tableau 4 Paramètres de stockage des dossiers iFolder stockés sur le serveur iFolder

Paramètre

Description

Disponible

Espace disque total disponible sur le serveur pour les fichiers ou les répertoires dans le dossier iFolder sélectionné. La valeur est égale au quota moins l'espace utilisé.

Utilisé

Espace disque total occupé sur le serveur par l'ensemble des fichiers et des répertoires dans le dossier iFolder sélectionné.

Quota

La valeur indique l'espace disque total du serveur alloué au dossier iFolder sélectionné par son propriétaire. Si aucun quota n'est défini, la valeur est vide ou N/A(non applicable).

Les paramètres suivants indiquent l'état de la synchronisation de la copie locale du dossier iFolder :

Tableau 5 Paramètres de synchronisation des dossiers iFolder stockés localement

Paramètre

Description

Dernière synchronisation réussie

Heure de fin de la dernière synchronisation de la copie locale du dossier iFolder avec le serveur iFolder.

mm/dd/yyyy hh:mm:ss

Fichiers/dossiers à synchroniser

Nombre total d'éléments de stockage (fichiers et dossiers) dans le dossier iFolder devant être synchronisés.

Synchronisation automatique tous les :

Intervalle de synchronisation du dossier iFolder. Spécifiez la valeur en minutes, heures ou jours.

Lorsque le client iFolder est utilisé avec un serveur d'entreprise Novell iFolder 3.6 , l'intervalle ne peut pas remplacer l'intervalle minimal spécifié par l'administrateur iFolder pour le serveur ou votre compte sur le serveur, mais il peut remplacer l'intervalle de synchronisation du client.

Synchronisation sécurisée

Permet une synchronisation sécurisée des fichiers du dossier iFolder que vous êtes en train de créer. Vous pouvez activer un canal sécurisé pour les dossiers iFolder ordinaires et codés. Lorsque cette option est sélectionnée, iFolder vous permet d'utiliser un canal sécurisé pour effectuer la synchronisation entre les dossiers iFolder sur le serveur et sur la machine locale.

Synchroniser maintenant

Cliquez sur ce bouton pour lancer une synchronisation sur demande.

Accès aux propriétés iFolder à partir d'un gestionnaire de fichiers

  1. Dans le gestionnaire de fichiers, accédez au dossier iFolder et sélectionnez-le.

  2. Cliquez avec le bouton droit sur le dossier iFolder, puis cliquez sur iFolder > Propriétés.

Accès aux propriétés iFolder à partir du navigateur iFolder

  1. Dans la zone Notification, cliquez avec le bouton droit sur l'icône Services iFolder, puis cliquez sur les dossiers iFolder pour ouvrir le navigateur iFolder.

  2. Dans la liste des dossiers iFolder, sélectionnez le dossier iFolder, puis cliquez sur Propriétés.

Pour plus d'informations, reportez-vous à la rubrique Mentions légales.
