iFolder 客户程序

iFolder 客户程序可与您的操作系统集成以在本机桌面环境中提供 iFolder 服务。通过使用 iFolder 客户程序，您可以管理自己的 iFolder，并且可以选择要在每台计算机上设置的共享 iFolder。iFolder 支持以下操作系统：

  • SUSE Linux Enterprise Desktop (SLED) 10 、SP3

  • SUSE Linux Enterprise Desktop (SLED) 11 SP2 64 位

  • openSUSE 12.1

  • Windows XP SP3（32 位）

  • Windows Vista SP2

  • Windows 7 SP1

  • Macintosh OS X 32 位（Intel 体系结构）v10.6.7 及更高版本（需要 Mono 2.4.2.3）。不支持 PowerPc 结构。

有关更多信息，请参见 法律声明.
